# Claude Code Rust - 快速开始指南

## 🚀 立即开始使用

### 1. 关闭之前运行的程序
如果之前启动过 Rust Claude，请先关闭它。

### 2. 编译项目
```powershell
cd C:\Users\user\claude-code-rust
cargo build --release
```

### 3. 启动 CLI 版本（推荐）
```powershell
C:\Users\user\claude-code-rust\target\release\claude-code.exe
```

### 4. 或启动 GUI 版本
```powershell
C:\Users\user\claude-code-rust\target\release\claude-code-gui.exe
```

---

## ✨ 新功能体验

### 📝 流式输出 + 打字机效果
当你发送消息后，Claude 的回复会**逐字符显示**，就像真正的打字一样！

### 🛠️ 使用工具
试试这些命令，Claude 会自动使用工具：
- "读取 README.md 文件"
- "显示当前目录的文件列表"
- "查看 git 状态"
- "搜索所有包含 'fn main' 的文件"

### 💬 REPL 命令
在交互模式中输入：
```
help      - 显示所有命令
tools     - 查看可用工具
history   - 查看对话历史
status    - 查看当前状态
reset     - 重置对话
clear     - 清屏
exit      - 退出
```

## 🔧 配置

### 查看当前配置
```powershell
C:\Users\user\claude-code-rust\target\release\claude-code.exe config show
```

### 设置 API Key
```powershell
C:\Users\user\claude-code-rust\target\release\claude-code.exe config set api_key "your-key"
```

### 同步 CC Switch 配置
```powershell
cd C:\Users\user\.claude-code
powershell -ExecutionPolicy Bypass -File sync-simple.ps1
```
