## 快速开始

### 1. 启动炫酷 CLI 界面

```bash
# 进入项目目录
cd C:\Users\user\claude-code-rust

# 启动 REPL（推荐）
.\target\release\claude-code.exe repl

# 或者直接运行
.\target\release\claude-code.exe
```

### 2. 常用命令

```bash
# 查看帮助
.\target\release\claude-code.exe --help

# 查看版本
.\target\release\claude-code.exe --version

# 查看当前配置
.\target\release\claude-code.exe config show

# 单次查询（非交互式）
.\target\release\claude-code.exe query --prompt "你好"
```

### 3. REPL 内命令

启动 REPL 后，可以使用以下命令：

```
help      - 显示帮助信息
status    - 查看状态信息
config    - 查看配置
history   - 查看对话历史
reset     - 重置对话
clear     - 清屏
exit      - 退出
```

## 配置修改

### 修改 API Key

```bash
.\target\release\claude-code.exe config set api_key "your-api-key"
```

### 修改模型

```bash
.\target\release\claude-code.exe config set model "Doubao-Seed-2.0-Code"
```

### 修改 Base URL

```bash
.\target\release\claude-code.exe config set base_url "https://ark.cn-beijing.volces.com/api/v3"
```

### 重置为默认配置

```bash
.\target\release\claude-code.exe config reset
```

## 多 API 路由配置

如果你需要切换不同的 API，可以使用环境变量：

### DeepSeek

```powershell
# 设置环境变量
$env:DEEPSEEK_API_KEY="sk-..."
$env:API_BASE_URL="https://api.deepseek.com"

# 然后运行
.\target\release\claude-code.exe
```

### 火山引擎（当前）

```powershell
# 确保使用配置文件中的设置
Remove-Item Env:\DEEPSEEK_API_KEY -ErrorAction SilentlyContinue
Remove-Item Env:\API_BASE_URL -ErrorAction SilentlyContinue

# 运行
.\target\release\claude-code.exe
```

## 故障排查

### API 请求失败

1. 检查网络连接
2. 确认 API Key 是否正确
3. 检查 Base URL 是否为 `https://ark.cn-beijing.volces.com/api/v3`
4. 查看当前配置：`.\target\release\claude-code.exe config show`

### 查看详细日志

```powershell
$env:RUST_LOG="debug"
.\target\release\claude-code.exe
```
